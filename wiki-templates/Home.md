## âœ¨ What is React InScroll?

React InScroll is a modern, performant scroll animation library for React applications. It provides 25+ animation types with TypeScript support and uses the Intersection Observer API for optimal performance.

- ğŸ­ **25+ Animation Types** - Fade, Zoom, Flip, Slide, and more
- ğŸ¯ **TypeScript Ready** - Full type safety out of the box
- âš¡ **Performance Optimized** - Uses Intersection Observer API

- ğŸ“± **Responsive** - Works seamlessly on all devices
- ğŸ”„ **Flexible Behavior** - One-time or repeating animations
